# styles module

in neo-jquery now adds core styles module inspired dorian designs videos

## using

``` html
<link rel="stylesheet" href="https://unpkg.com/browse/neo-jquery@1.0.278/css/neo-jquery.css" />
```

## layout class

``` html
<section class="layout">
  <header>...</header>
  <article>hello world</article>
  <footer>..</footer>
</section>
```

<section style={{
  display: 'grid',
  minHeight: '100vh',
  gridTemplateRows: 'auto 1fr auto'
}}>
      <header style={{background: '#000', color: '#fff'}}>
        <h2>example example</h2>  
      </header>
      <article style={{background: '#0f0',}}>hello world</article>
      <footer style={{background: '#000', color: '#fff'}}>example footer</footer>
</section>
