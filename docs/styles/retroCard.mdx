# retro card docs

new styles module for neo-jquery added in version 1.1.0 using css no js using

## stylesheet

``` html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/neo-jquery/css/retro-card.css">
```

## example

``` html
<section class="retro-card">
  <h2>hello</h2>
  
  <p>lorem ipsum</p>
</section>
```
