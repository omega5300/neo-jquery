# tooltip module

new styles module for neo-jquery added in version 1.0.4 using css no js using

## stylesheet

``` html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/neo-jquery/css/tooltip.css">
```

## example

``` html
<span data-tooltip="example" class="tooltip">Tooltip text</span>
```
